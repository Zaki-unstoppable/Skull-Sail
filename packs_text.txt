Skull & Sail  Merged Claude Code Prompt Packs  Progression-aware merged prompts with Babylon.js-specific leverage added where it materially improves implementation.  How to use this document  • Read the current repo and relevant branch files first. Treat the uploaded code as the source of truth.  • Stay delta-only. Do not re-implement systems that already exist unless a pack explicitly revises them.  • These packs remain progression-aware, but they are modular: later packs can follow earlier ones  without requiring every prior pack to be 100% finished.  • Babylon-specific features are called out only where they meaningfully improve the result, such as GLB  workflows, PBR/environment lighting, nav mesh, animation/retargeting, particles/VFX, Inspector, and  the audio engine.  Merge rule used here:   prompts were merged only when the combined prompt stayed clear enough for Claude Code while preserving the original implementation asks, constraints, and validation/tuning requirements.
